# EntregaExpress_P2

## Puertos de los microservicios

Los puertos que se usarán para ejecutar los microservicios en desarrollo:

- **api-gateway:** 8080
- **auth-service:** 8081
- **billing-service:** 8082
- **fleet-service:** 8083
- **pedido-service:** 8084

## Puerto de PostgreSQL (Docker)

El contenedor de PostgreSQL usará el puerto por defecto:

- **PostgreSQL (host:container):** 5432:5432

> Nota: Si alguno de estos puertos ya está en uso en tu máquina, cámbialos por otros libres y actualiza las configuraciones correspondientes.
